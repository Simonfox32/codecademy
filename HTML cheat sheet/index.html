HTML Boilerplate Cheat Sheet

HTML elements

DOCTYPE html

<!DOCTYPE html> tells the browser which HTML version the site was written in. Without this declaration, the browser enters "Quirks Mode". Quirks mode causes rendering issues, performance problems, and JavaScript incapatability. Be thankful for <!DOCTYPE html> and don't forget to include it in the start of your index file!

HTML

<html> tells the browser to read the file as an HTML file. Without the tag the browser would be incapable of reading your HTML file. <html> goes right under the DOCTYPE tag.

Head

<head> is a container used to hold important elements about the page that are not rendered. Some of these elements include <meta, and <link, <title>. Head is where CSS stylesheets are linked so make sure not to forget it!

Important Elements Within Head

Tag Name Description
<meta charset="utf-8"/> Meta Charset Renders Characters!
<meta name="viewport"/> Meta Viewport Optimizes page for mobile!
<link/> Link Links to stylesheet!
<title> Title Specifies website title!

Body

<body> is a container used to hold every element that is rendered to the page. The majority of HTML is written in this container. Make sure to include it or else nothing else will be rendered!
